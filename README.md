Legado Mesozoico Argentino 🦖
​1. Descripción del Proyecto
​Legado Mesozoico Argentino es una aplicación web educativa e interactiva diseñada para explorar y organizar el registro fósil de especies halladas en territorio argentino. La plataforma permite a los usuarios acceder a un catálogo detallado de especímenes, consultar información científica precisa y gestionar la colección desde un panel administrativo dedicado.

​3. Funcionalidades Principales
​Catálogo Interactivo: Visualización de especímenes con detalles sobre su morfología, época y relevancia paleoecológica.
​Autenticación: Sistema de acceso restringido para asegurar la integridad de la gestión.
​Panel Administrativo: Interfaz protegida para la creación, edición y eliminación de registros (CRUD) de dinosaurios.
​Búsqueda y Filtrado: Herramientas para localizar especies según criterios específicos.
​4. Tecnologías Utilizadas
​Frontend: React (con hooks, lazy loading y Zustand para el manejo del estado global).
​Enrutamiento: Wouter.
​Formularios: React Hook Form con validación mediante Zod.
​Comunicación: Axios para el consumo de la API REST propia.
​Estilos: CSS modularizado e integrado con la estética de bitácora antigua.
