# 🦖 Legado Mesozoico Argentino

## 📖 Descripción

**Legado Mesozoico Argentino** es una aplicación web educativa e interactiva destinada a explorar y organizar el registro fósil de especies descubiertas en territorio argentino.

La plataforma permite consultar un catálogo de dinosaurios con información científica detallada sobre cada espécimen, además de ofrecer un panel administrativo protegido para gestionar la colección de manera segura.

## ✨ Funcionalidades

### 📚 Catálogo Interactivo
- Visualización de especies fósiles.
- Información sobre morfología.
- Época geológica.
- Relevancia paleoecológica.

### 🔐 Autenticación
- Inicio de sesión para usuarios autorizados.
- Protección del acceso al panel administrativo.

### ⚙️ Panel Administrativo
Operaciones CRUD completas para la gestión de especies:

- ➕ Crear registros.
- ✏️ Editar información.
- 🗑️ Eliminar registros.
- 👀 Visualizar especies.

### 🔎 Búsqueda y Filtrado
- Búsqueda por nombre.
- Filtrado según diferentes criterios para facilitar la exploración del catálogo.
